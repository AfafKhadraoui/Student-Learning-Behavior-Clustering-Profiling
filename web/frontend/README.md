# Frontend

This package contains the React dashboard for the clustering project.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Recharts and Radix UI for the UI layer

## Purpose

The frontend renders cluster summaries, per-student views, and analysis charts using data exposed by the backend API. It is not responsible for training models or preprocessing the dataset.

## Notes

- `src/` holds the application code.
- `public/` contains static assets.
- Keep the UI focused on the project outputs generated by the notebooks and backend.
