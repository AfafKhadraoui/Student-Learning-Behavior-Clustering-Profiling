# Overleaf report bundle

This folder contains a LaTeX draft of the full project report.

## Files
- `main.tex` — full report draft
- `references.bib` — bibliography entries

## How to use in Overleaf
1. Create a new Overleaf project.
2. Upload `main.tex` and `references.bib`.
3. Upload the `figures/` folder from the repository root if you want the plots to render.
4. Compile with BibTeX.

## Notes
- The report uses the model comparison export from `reports/results/model_comparison_summary.csv`.
- Relative figure paths are configured for both local and Overleaf-style folder layouts.
